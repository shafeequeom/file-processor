# File processor

## About

File Processor
## Features

- File Upload
- Analytics

## Installation

#### yarn

```sh
yarn install
```

#### npm

```sh
npm install
```

## Running application

For development environment...
Using YARN:

```sh
yarn dev
```

Using NPM:

```sh
npm run dev
```

For production environment...
Using YARN:

```sh
yarn start
```

Using NPM:

```sh
npm start
```
